# Evaluate a News Article with Natural Language Processing

## Overview
This is the Udacity Project I created for evaluating a News Article details.

## Instructions
User will enter the url or the text and based on that the app will display the model, agreement & subjectivity of the text in the url or the entered text on clicking the submit button.

## Extras
meaningcloud.com/ has been referred to generate the output using its API https://api.meaningcloud.com/sentiment-2.1 along with the api key.
